# Virtual Piano ESP32 Project

This project contains two separate Arduino ESP32 sketches:

1. instrument_config/instrument_config.ino
   - UI board
   - TFT touch screen
   - instrument icons
   - DFPlayer control

2. SENS_FIXED/SENS_FIXED.ino
   - sensor board
   - ToF sensors
   - HC-12 communication

Important:
Do not merge the two .ino sketches into one Arduino sketch.
