## תקציר מצב נוכחי

הפרויקט כולל שתי סקיצות Arduino נפרדות:

- `instrument_config/instrument_config.ino` - לוח UI עם מסך מגע, אייקונים ושליטת `DFPlayer`.
- `SENS_FIXED/SENS_FIXED.ino` - לוח חיישנים עם חיישני ToF ותקשורת `HC-12`.

אין למזג את הסקיצות. כל סקיצה צריכה להיבנות, להיטען ולהיבדק בנפרד.
