## כלל מיפוי

יש לוודא שמיפוי הקבצים בכרטיס ה-SD תואם ללוגיקת `DFPlayer` בסקיצת ה-UI.

דוגמה למיפוי אפשרי:

`track = instrumentIndex * 8 + noteIndex + 1`
