## סיכום מסירה

המערכת כוללת:

- לוח UI עם סקיצת `instrument_config/instrument_config.ino`.
- לוח Sensor עם סקיצת `SENS_FIXED/SENS_FIXED.ino`.
- מסך מגע עם `TFT9341Touch`.
- קבצי אייקונים `.h` בתיקיית `instrument_config/`.
- תקשורת בין הלוחות דרך `HC-12` או Serial תואם.
- השמעת שמע דרך `DFPlayer`.
- חיישני ToF דרך `VL53L0X`, `VL6180X` ו-`TCA9548A`.
