# טפסים לניהול הפרויקט

תיקייה זו כוללת טפסים מוכנים למילוי עבור ניהול, בדיקה ואישור של פרויקט ESP32 Virtual Piano.

## בעלי תפקידים

- בעל הפרויקט: מגדיר מטרות, מאשר דרישות ושינויים.
- בודק תוכנה: בודק קומפילציה, התנהגות קוד, תקשורת ובדיקות אינטגרציה.
- בודק אלקטרוניקה/חיווט: בודק רכיבים, חיבורים, מתח, פינים ויציבות חשמלית.
- מורה/מנחה: בודק התקדמות, התאמה לדרישות ויכולת הצגה.
- מעריך הדגמה סופית: מאשר שהמערכת עובדת בהצגה הסופית.

## הנחיית שימוש

יש לעדכן את הטפסים לאחר כל שינוי טכני, במיוחד שינויי חיווט, כיול, מיפוי קבצי קול, תיקוני קומפילציה ושינויי דרישות.
